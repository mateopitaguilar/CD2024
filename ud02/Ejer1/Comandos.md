Crear archivos

    touch mateo.txt actividades.txt

Modificar el contenido del archivo

    nano mateo.txt
    echo "Soy Mateo y estudio DAW" >> mateo.txt

Hacer Commit

    git commit -m "Mensaje commit"

Añadir todos los cambios de la carpeta

    git add .

Añadir un archivo en especifico al commit 

    git add mateo.txt

Crear una carpeta

    mkdir carpeta1

Crear fichero

    touch fichero1.txt fichero2.txt

Volver a una version anterior

    git checkout (nombre/codigo del commit)

Volver a la principal

    git checkout [nombre de la rama principal(main)]
